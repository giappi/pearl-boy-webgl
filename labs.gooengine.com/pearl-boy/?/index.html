Pearl Boy was created with Goo Engine, a 3D engine based on HTML5/WebGL.
Goo Create is a visual editor for making 3D apps and games and publishing them on the web.
Click here to get started!
Pearl Boy was created to show the possibilities of Goo Engine and HTML5/WebGL. By using the performance of WebGL and hardware accelerated graphics, combined with our javascript game engine, we can provide rich 3D content in the browser. It will work cross-platform on any WebGL-enabled device and there is no need for plugins like Flash or Silverlight.
